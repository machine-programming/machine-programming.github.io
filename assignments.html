Assignments

Assignment Policy: All assignments are released when the course starts and can be submitted through GradeScope anytime before the deadline. Late submissions will be penalized 10% per day up to 5 days. Collaboration is allowed, but you must cite your collaborators in the submission, and the submission must be your own work.

Extensions are allowed, but you must request an extension in advance via email to the instructor (ziyang@cs.jhu.edu). The maximum extension is 2 days. No extensions will be granted for late submissions.

Each enrolled and auditing student is granted an API key for a large language model for the assignments. You can use the API key to access the language model in your assignments; the code that setup the client is provided in the assignment skeleton code (Assignment 2 and Assignment 3).
